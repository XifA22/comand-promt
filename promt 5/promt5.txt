Percobaan pertama
buatkan aplikasi streamlit, pada halaman pertama header terdapat tulisan visualisasi data, kemudian terdapat 3 tombol dengan tulisan lingkaran, batang, garis. jika tombol lingkaran diklik menampilkan halaman baru dengan fitur tombol upload csv, tombol pengertian, tombol panduan penggunaan, dan menampilkan gambar visualisasi data. jika tombol upload csv diklik bisa upload file format csv dan mengekstrasi nama kolom pertama sebagai nama irisan kemudian memberi format angka pada sumbu y. untuk tombol batang dan garis sama dengan tombol lingkaran fiturnya
