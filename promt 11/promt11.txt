Percobaan pertama
Buat aplikasi streamlit dengan halaman utama berisi logo dipojok kiri atas dan teks visualisasi data berada diatas tombol dan tiga tombol bertuliskan diagram lingkaran batang dan garis. pada lingkaran ketika tombol ini diklik, buka halaman baru dengan fitur upload, ekstraksi kolom pertama sebagai nama irisan, format angka pada sumbu Y, tombol Pengertian, tombol Panduan Penggunaan dan gambar visualisasi. pada batang baca data dari CSV, ekstraksi kolom pertama,tombol Pengertian, tombol Panduan Penggunaan dan gambar visualisasi, tampilkan tahun, buat diagram batang vertikal dengan tombol Pengertian, tombol Panduan Penggunaan dan gambar visualisasi, label, grid, dan angka detail di atas batang. pada garis tampilkan uploader, tombol syarat dan setentuan, baca data dari CSV, format angka pada sumbu Y, buat diagram garis dengan angka detail, dan grid. Pisahkan kode ke dalam empat file yaitu main.py untuk halaman utama, diagram_lingkaran.py untuk diagram lingkaran, diagram_batang.py untuk diagram batang, dan diagram_garis.py untuk diagram garis.
