Percobaan pertama
buatkan aplikasi streamlit, pada halaman pertama header terdapat tulisan visualisasi data, kemudian terdapat 3 tombol dengan tulisan lingkaran, batang, garis. jika tombol lingkaran diklik menampilkan halaman baru dengan fitur tombol upload csv, tombol pengertian, tombol panduan penggunaan, dan menampilkan gambar visualisasi data. jika tombol upload csv diklik bisa upload file format csv dan mengekstrasi nama kolom pertama sebagai nama irisan kemudian memberi format angka pada sumbu y. untuk tombol batang bisa Membaca data dari file CSV yang diunggah, Ekstraksi nama kolom pertama, Ekstraksi data dari kolom pertama, Menampilkan daftar tahun yang tersedia, Ekstraksi jumlah penduduk untuk tahun yang dipilih, Membuat diagram batang vertikal, Menambahkan judul dan label,  Menambahkan grid, Menambahkan angka detail di atas batang, Menampilkan diagram batang vertikal, Menambahkan judul dan label, Menambahkan grid, Menambahkan angka detail di samping batang. untuk diagram garis Menampilkan komponen file uploader, Menambahkan tombol Syarat dan Ketentuan, Membaca data dari file CSV yang diunggah, Ekstraksi data dari dataframe, Fungsi untuk format angka pada sumbu y, Membuat diagram garis, Menambahkan angka detail di atas titik data, Menambahkan grid dan format angka pada sumbu y, Menampilkan diagram. buat jadi 4 file, file pertama main.py berisi halaman pertama yang ada 3 tombol tadi, file kedua berisi diagram lingkaran, file ketiga berisi diagram batang, file keempat berisi diagram garis
