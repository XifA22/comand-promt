Percobaan pertama
Buat aplikasi streamlit dengan halaman utama berisi logo dipojok kanan atas dan teks "visualisasi data" berada diatas tombol dan tiga tombol bertuliskan diagram lingkaran, batang dan garis lalu jika diklik menuju halaman baru, pada halaman baru ada sidebar dan tombol jika diklik Kembali ke halaman utama. pada diagram lingkaran mempunyai fitur upload file csv, ekstraksi kolom pertama sebagai nama irisan, format angka pada sumbu Y, tombol Pengertian jika tombol pengertian diklik muncul teks, tombol Panduan Penggunaan jika diklik muncul teks dan gambar, dan gambar visualisasi. pada diagram mempunyai fitur upload file csv, ekstraksi kolom pertama, tampilkan tahun, buat diagram batang vertikal dengan judul, label, grid, dan angka detail di atas batang. pada garis tampilkan uploader, tombol syarat dan setentuan, baca data dari CSV, format angka pada sumbu Y, buat diagram garis dengan angka detail, dan grid. Pisahkan kode ke dalam empat file yaitu main.py untuk halaman utama, diagram_lingkaran.py untuk diagram lingkaran, diagram_batang.py untuk diagram batang, dan diagram_garis.py untuk diagram garis.
